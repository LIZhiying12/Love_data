# Love_data
微信聊天记录可视化教程

所用编程语言与软件   python+pycharm

环境配置
python   3.10.9
matplotlib  由于包不兼容问题，使用两个版本  3.7.0/3.8.2  代码中有解释
下列包下载最新版即可
numpy
pandas
seaborn
jieba
july
wordcloud

步骤一
通过github开源项目“留痕”解包聊天记录，得到聊天记录csv文件，详细教程参考https://github.com/LC044/WeChatMsg
该项目也可进行简单数据分析，无代码能力可直接使用

步骤二
编写python代码进行数据可视化
